# Properties Management Website

A modern and fully responsive website for **Properties Management**, a company specializing in **property and facility services** across **Algeria**, **Morocco**, and **Tunisia**.

The platform showcases a wide range of services including:
- Real estate management
- Interior workspace development
- Cleaning and sanitization
- Events and catering
- Landscaping, HVAC, security, transport, and more
